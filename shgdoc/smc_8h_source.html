44  * First we introduce sojourn time distributions. Struct STD is
45  * used to derive five classes of these distributions:
46  * 1. distribution with finite number of values,
47  * 2. geometric distribution,
48  * 3. negative binomial distribution,
49  * 4. logarithmic series distribution,
50  * 5. %Poisson distribution.
51  *
52  * Because negative binomial distribution and %Poisson
53  * distribution both accept the value 0, at the moment the value

55  * additional integer parameters A > 0 and B > 0 for %Poisson

118  /**
119  * Generates T-element SMC if cut == true. If cut == false,
120  * generates at least T elements to end the sample at the end of
121  * the last sojourn time.
122  *
123  * Returns: 0 - ok, 1 - invalid dimension of alpha_ or P_, 2 -
124  * incorrect value of alpha_, 3 - incorrect value of P_, 4 - std_
125  * not assigned, 5 - g_ not assigned, 6 - T is not positive.
126  *
127  * The state space is {0, 1, ..., s_ - 1}. On successful return:
128  *
129  * Z[t], 0 <= t < T, is the generated SMC,
130  *
131  * X.size() == S.size() == J.size()
132  *
133  * The process entered the i-th successive state J[i] at the
134  * moment S[i] and remained in this state during successive X[i]
135  * moments. S[0] = 0, S[i] = S[i - 1] + X[i - 1] for i >= 1, Z[t]
136  * = J[i] for t = S[i], ..., S[i] + X[i] - 1, i = 0, ...,
137  * X.size().
138  */
