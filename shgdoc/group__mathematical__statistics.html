Kolmogorov-Smirnov test.

Given a cumulative continuous distribution function cdf and a vector of data x, the function performs Kolmogorov-Smirnov test and returns the Kolmogorov-Smirnov statistic D d and its p-value prob. Small values of prob indicate that the cumulative distribution function of data is significantly different from cdf.

Warning
After return, x is sorted non-decreasingly.
Implementation
This function is a modification of the function ksone from [33], p. 737, to allow function objects as parameters. We do not use approximation (14.3.18) from [33], p. 737. See also [42], p. 306.
