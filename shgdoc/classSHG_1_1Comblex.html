Detailed Description

Generates all k-combinations of an n-set in lexicographic oreder.

There must be 0 < k <= n. The number of combinations is

Example program:
/* comblex.cc: generates 3-element combinations of 4-element set */
/*
* Output:
*
* 012
* 013
* 023
* 123
*/
#include <iostream>
#include "shg/utils.h"
int main() {
const int n = 4, k = 3;
SHG::Comblex c(n, k);
do {
const std::vector<int>& v = c.get();
for (int i = 0; i < k; i++)
std::cout << v[i];
std::cout << '\n';
} while (c.next());
}
