22 /**
23  * Ordinal patterns distribution in time series.
24  *
25  * Let \f$(x_i)_{i = 0}^{n - 1}\f$ be a sequence of elements. A
26  * subsequence \f$x_i, x_{i + 1}, \ldots, x_{i + d - 1}\f$ of length
27  * \f$d > 1\f$ defines the ordinal pattern \f$\pi = (\pi_0, \pi_1,
28  * \ldots, \pi_{d - 1})\f$ if \f[x_{i + \pi_0} < x_{i + \pi_1} <
29  * \ldots < x_{i + \pi_{d - 1}},\f] and \f$x_k < x_l\f$ if \f$x_k =
30  * x_l\f$ but \f$k < l\f$ \cite amigo-2010, p. 52.
31  *
32  * The constructor accepts the sequence \f$(x_i)_{i = i_0}^{i_1 -
33  * 1}\f$ and the block length \f$d > 1\f$. Then the map frequency
34  * returns the distribution of ordinal patterns of length \f$d\f$
35  * existing in the sequence.
36  */

53  * The ordinal pattern \f$\pi = (\pi_0, \pi_1, \ldots, \pi_{d -
54  * 1})\f$ corresponds to the condition \f[x_{i + \pi_0} < x_{i +
55  * \pi_1} < \ldots < x_{i + \pi_{d - 1}},\f] wherein if \f$\pi_k

105  pattern p(d);
106  for (sztp i = 0; i < imax; i++) {
