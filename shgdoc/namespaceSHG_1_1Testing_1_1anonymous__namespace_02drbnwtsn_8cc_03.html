void SHG::Testing::anonymous_namespace{drbnwtsn.cc}::cdftst ( )

Compares the results of calls to dwcdf with those read from the file drbnwts1.txt.

If the difference is too big, an SHG_ASSERT(0) is executed. The results in the input file come from the function dwcdf itself and were checked to be equal to the results of the function nag-prob-durbin-watson (g01epc) from the NAG C library, mark 7 (2001).
