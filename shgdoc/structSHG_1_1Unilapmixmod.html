Detailed Description

Univariate Laplace mixture models.

Univariate Laplace mixture model estimation by expectation-maximization algorithm. After initialization with data vector x and the number of components K, user must set initial values of Unilapmixmod::pi, Unilapmixmod::mu, Unilapmixmod::lambda, call estep() and mstep(). Then estep() and mstep() must be called alternately until the difference between two consecutive values of Unilapmixmod::loglik returned by estep() is sufficiently small.

Implementation
The class is modeled after the class Gaumixmod [33], p. 845, and based on [25], p. 359.
