int SHG::SMC::generate ( int  T,
bool  cut = false 
)

Generates T-element SMC if cut == true.

If cut == false, generates at least T elements to end the sample at the end of the last sojourn time.

Returns: 0 - ok, 1 - invalid dimension of alpha_ or P_, 2 - incorrect value of alpha_, 3 - incorrect value of P_, 4 - std_ not assigned, 5 - g_ not assigned, 6 - T is not positive.

The state space is {0, 1, ..., s_ - 1}. On successful return:

Z[t], 0 <= t < T, is the generated SMC,

X.size() == S.size() == J.size()

The process entered the i-th successive state J[i] at the moment S[i] and remained in this state during successive X[i] moments. S[0] = 0, S[i] = S[i - 1] + X[i - 1] for i >= 1, Z[t] = J[i] for t = S[i], ..., S[i] + X[i] - 1, i = 0, ..., X.size().
