43 /**
44  * Class Data provides an array of prices of assets. Layout of data is
45  * as follows:
46  *
47  * <pre>
48  * |-----------------------------------------------------------|
49  * | j | 0 1 ... nobs_ - 1 |
50  * | date_[j] | 19910416 19910417 ... 20090414 |
51  * | i name_[i] | |
52  * |-------------------------+---------------------------------|
53  * | 0 | 01NFI | |
54  * | 1 | 04PRO | |
55  * | ... | ... | price_[i][j] |
56  * | nassets_ - 1 | ZYWIEC | |
57  * |-----------------------------------------------------------|
58  * </pre>
59  *
60  * Data::nassets_ is the number of stocks, Data::nobs_ is the number
61  * of trading days in the data set. The vectors Data::name_ and
62  * Data::date_ are sorted: name_[0] < ... < name_[nassets_ - 1],
63  * date_[0] < ... < date_[nobs_ - 1]. The matrix Data::price_ is a
64  * matrix of Data::nassets_ rows and Data::nobs_ columns of double.
65  * Data::price_[i][j] is a price of the stock Data::name_[i] quoted at
66  * Data::date_[j]. For missing values Data::price_[i][j] == -1.
67  *
68  * \implementation Justification of the method. Within the period from
69  * 16 April 1991 to 15 October 2010, 521 stocks were quoted. Each
70  * quotation requires 33 bytes. On a 32-bit system there should be no
71  * difficulty to allocate 2^31 - 1 = 2147483647 bytes. If the number
72  * of stocks were four times greater (2084) and the file were
73  * collected 100 years 260 trading days a year, the file would contain
74  * 2084 x 260 x 100 x 33 = 1788072000 bytes, that is 83% of 2^31 - 1.
75  */

154 /**
155  * Gets sample of historical observations on rates of return for
156  * experiment consisting in buying assets on day d and selling them on
157  * day d + frequency. We examine the period
158  *
159  * (d - frequency - reserve) (d - frequency - reserve + 1) ... (d - 1)
160  *
161  * Selected assets must be quoted on day d and on day d + frequency.
162  *
163  * The return is valid if nassets >= 1. Then nobs >= 1, a contains,
164  * indexes of assets from name() and r contains sample rates of
165  * return, r[i][j], 0 <= j < nobs, being observations on rates of
166  * return on assets named name()[a[i]], i <= 0 < nassets. If on exit
167  * nassets == 0, no sample has been found, and values of nobs, a and r
168  * are indefinite.
169  *
170  * The variable reserve is a reserve of days. The greater reserve, the
171  * more observations will be used to calculate moments. For example,
172  * if reserve = 90 and frequency = 1, we calculate rates of return
173  * using data from three months just before start of investment.
174  */
175 void msample(const Data& d, Sample_data& sd);

192 /**
193  * This class selects nseries series of rates of return. The last data
194  * on prices may be from last_date. First, the assets which have more
195  * than max_missing missing observations are removed. Second, the
196  * remaining assets are sorted by the number of observations and the
197  * first nseries is selected with the greatest number of observations.
198  * These assets are than sorted alphabetically. The missing
199  * observations on prices are linearly interpolated, then rates of
200  * return are calculated. If replace_exact_0 is true, all rates of
201  * return that are equal to 0.0 are replaced by random normal numbers
202  * with mean 0.0 and small variance using rng generator (if
203  * replace_exact_0 is false, rng is not used and may be 0). Then data
204  * is output to the file ofn. A summary is printed to the file lfn.
205  * The summary contains:
206  *
207  * 1. Number.
208  * 2. Name of the stock.
209  * 3. %Date of the first observed rate of return.
210  * 4. %Date of the last observed rate of return.
211  * 5. The number of observations in the series of rates of return.
212  * 6. The number of observations missing in the series of prices. The
213  * series of prices has one observation more than the series of rates
214  * of return. These observations were linearly interpolated.
215  * 7. The number of rates of return which were exactly equal to zero
216  * and were replaced by random normal numbers.
217  * 8. Percentage of replaced observations.
218  */
