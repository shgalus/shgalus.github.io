19 /**
20  * Cumulative distribution function of the Durbin-Watson statistic.
21  *
22  * \f$n \geq 1\f$ denotes the number of observations, \f$k \geq 1\f$
23  * is the number of regressors excluding the intercept, \f$x\f$ is the
24  * point at which the function should be evaluated,
25  * \f$\mathit{lower}\f$ should be \f$\mathit{true}\f$ for \f$d_L\f$
26  * and \f$\mathit{false}\f$ for \f$d_U\f$, \f$\mathit{eps} > 0\f$ is
27  * the accuracy for calculating the result (see equation (4) on page 2
28  * in <a href="drbnwtsn.pdf">Cumulative distribution function of the
29  * Durbin-Watson statistic.</a>), \f$\mathit{iter} \geq 1\f$ is the
30  * maximum number of iterations in the Simpson method. In the
31  * \f$i\f$-th iteration, \f$1 \leq i \leq \mathit{iter}\f$, \f$2^i\f$
32  * new values of the integrand are calculated. The total number of
33  * evaluations of integrand is \f$2^{\mathit{iter} + 1} + 1\f$. There
34  * must be \f$n - k - 1 > 1\f$.
35  *
36  * \return the value of the cumulative distribution function
37  *
38  * \exception std::invalid_argument if \f$n \leq 0\f$, \f$k \leq 0\f$,
39  * \f$n - k - 1 \leq 1\f$, \f$\mathit{eps} \leq 0\f$ or
40  * \f$\mathit{iter} < 1\f$
41  *
42  * \exception std::range_error if the number of iterations would be
43  * greater than \f$\mathit{iter}\f$
44  *
45  * See <a href="drbnwtsn.pdf">Cumulative distribution function of the
46  * Durbin-Watson statistic.</a>
47  *
48  * \sa ppdw, swtbl
49  */

52 /**
53  * The percentage points of the Durbin-Watson statistic.
54  *
55  * \f$n \geq 1\f$ denotes the number of observations, \f$k \geq 1\f$ is
56  * the number of regressors excluding the intercept, \f$0 < p < 1\f$ is
57  * the probability for which percentage point shoul be evaluated,
58  * \f$\mathit{eps} > 0\f$ is the accuracy of calculating. There must be
59  * \f$n - k - 1 > 1\f$.
60  *
61  * On return, \f$x_L\f$ and \f$x_U\f$ contain values such that
62  * \f$\Pr(d_L < x_L) = p\f$ and \f$\Pr(d_U < x_U) = p\f$.
63  *
64  * The method of bisection is used twice to solve equations \f$\Pr(d_L <
65  * x_L) = p\f$ and \f$\Pr(d_U < x_U) = p\f$.
66  *
67  * \exception std::invalid_argument if \f$n \leq 0\f$, \f$k \leq 0\f$,
68  * \f$n - k - 1 \leq 1\f$, \f$p \leq 0\f$ or \f$ p \geq 1\f$
69  *
70  * \exception std::range_error if the number of iterations in dwcdfwould
71  * be greater than 15
72  *
73  * \sa dwcdf, swtbl
74  */
