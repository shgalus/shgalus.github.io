Detailed Description

A class to store a calendrical date.

The class is capable of storing dates from 1 January 1583 to 31 December 9999.

Date formats.

Four formats of date are supported. They are listed below with examples of the date 22 January 1901.

  • full "22 stycznia 1901"
  • Roman "22 I 1901"
  • common "22.01.1901"
  • computer "19010122"

The full format returns the month name in Polish in genitive case in UTF-8 encoding.

SHG::Date::Date ( const std::string &  s)
explicit

Constructs initialized date.

This constructor accepts all four formats named in Date. In full and Roman formats the number of the day must not start from 0. In common and computer formats the number of the day and the number of the month must have two digits. In all formats the number of the year must have 4 digits. Leading and trailing white space is not allowed.

Examples:

  • Date("22 stycznia 1901") — OK
  • Date("22 I 1901") — OK
  • Date("22.01.1901") — OK, length must be 10
  • Date("19010122") — OK, length must be 8
  • Date("02 stycznia 1901") — will fail
  • Date("02 I 1901") — will fail
  • Date("22.1.1901") — will fail
Exceptions
Bad_dateif the date is incorrect

SHG::Date::Date ( int  n,
int  d,
int  m,
int  y 
)

Constructs initialized date.

Constructs date in the following way:

  1. if n > 0 this is the n-th day of week d in month m in year y
  2. if n < 0 this is the -n-th day of week starting from the last day in the month m.

For example, to get date of the next to the last Friday in November 2011 call Date(-2, fri, nov, 2011).

Exceptions
Bad_dateon incorrect input
