Detailed Description

A class for making assertions.

It should be used by the means of the macros SHG_ASSERT(e) and SHG_THROW(). what() is always initialized to "assertion failed".

void SHG::Assertion::print ( const char *  progname = nullptr,
std::ostream &  f = std::cerr 
) const
virtual

If progname is given and not empty, it is printed followed by a colon and a space. Then what() is printed. If file() is not nullptr and not empty, " in file " file() ", line " line() is printed. Then the newline character is printed. For example, the code

Assertion("file.c", 10).print("prog");
Assertion("file.c", 10).print();
Assertion(nullptr, 10).print("prog");
Assertion(nullptr, 10).print();

produces the following output: 

* prog: assertion failed in file file.c, line 10
* assertion failed in file file.c, line 10
* prog: assertion failed
* assertion failed
*
