Detailed Description

Univariate Gaussian mixture models.

Univariate Gaussian mixture model estimation by expectation-maximization algorithm. After initialization with data vector x and the number of components K, user must set initial values of Unigaumixmod::pi, Unigaumixmod::mu, Unigaumixmod::sigma, call estep() and mstep(). Then estep() and mstep() must be called alternately until the difference between two consecutive values of Unigaumixmod::loglik returned by estep() is sufficiently small.

Implementation
The class is modeled after the class Gaumixmod [33], p. 845, and based on [25], p. 359.
